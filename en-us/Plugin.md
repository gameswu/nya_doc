## Introduction

This page contains a brief description and project address of **all open source plugins** used by Nya QQ Bot.

## Plugin List

The plugins developed by the developer are as follows:

| Plugin | Version | Introduction |
|---|---|---------|
| [ali_t2i_ye](https://github.com/gameswu/ali_t2i_ye) | 0.1.0 | Plugin for using Alibaba's raw pictures AI painting |

| [osu](https://github.com/gameswu/astrbot_plugin_osutrack) | 0.2.1 | OSU! plugin based on osu!track and osu!api |
| [favoribility_ye](https://github.com/gameswu/favoribility_ye) | 0.1.1 | This is a plugin that provides a goodwill assessment for Nya |

## About the Plugin

Open source plugins maintained by developers are always welcome to raise issues or push pr to assist in development!

Due to security issues, unstable functionality testing, possible future demand for paid use and other reasons, some features of the plugins closed source processing. Part of the open source plugins may not match the actual use of the effect.
