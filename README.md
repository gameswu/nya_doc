# 简介

> 喵\~ 欢迎使用小夜机器人！ 我是你的贴心小助手，随时为你提供帮助和陪伴喵\~ ——小夜

## 谁是小夜

小夜 QQ Bot 是基于 [AstrBot](https://astrbot.app/) 框架开发的大语言聊天机器人，开发初衷是作为开发者探索大语言应用以及角色扮演拟人化的玩具项目。2025年4月17日开发者首次公开部署在群聊内，而截止到2025年9月7日的晚上11点，小夜已经存在于87个群聊中，更是拥有300+的好友。:books:在[这里](DevStory.md)查阅更多开发幕后故事。

## 关于该文档

此前由于**意外的**传播，许许多多新朋友接触到了小夜，但是却对小夜的功能和特性不慎了解并引起了一些事件，最终导致了一些朋友对机器人的误解并对开发者造成了极大的困扰。

因此该文档主要用于帮助大家了解小夜的使用方法，回答大家关心的问题。同时也会分享关于小夜有意思的故事、里程碑以及部分开发者开发的历程。
